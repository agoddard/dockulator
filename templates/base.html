What's going on here?

When you enter a calculation in the box below (such as 1+4, 23.33/14.12 or -1000.0002/121.33) it gets added to a queue. There is a service watching the queue for new calculations. When that service picks up a new calculation, it assigns it a random operating system and a random programming language.

With the help of Docker, we're able to run the correct programming language on the correct OS and return you an answer all very quickly!
