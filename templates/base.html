Dockulator

When you enter a calculation in the box (such as 1 + 4, 23.33 / 14.12 or -1000.0002 / 121.33) it gets added to a queue. When a calculation is added to the queue it is picked up by a service watching the queue. The calculation is then assigned a random operating system and programming language. With the help of Docker, it figures out how and where to compute the given calculation. The calculation is added back to a completed list of calculations and that is what you see below.
